How to read your connection report

Built for Wi-Fi pros, useful for everyone

If you design, install, or troubleshoot Wi-Fi and networks for a living, this is the reference and calculator set you keep in your pocket, plus the active network tools you reach for in the field.

If you are not a Wi-Fi pro, Check My Connection still answers the one question you care about: is it the Wi-Fi or the internet?

What is inside

140 tools across 5 categories:

  • Calculators. RF and signal math without the spreadsheet: dBm and Watt, Free Space Path Loss, EIRP, link budget, fade margin, Fresnel zone, cable loss, antenna downtilt, throughput, PoE budget, capacity planning, subnet and IPv6 math, and more.
  • Reference tables. The tables you keep looking up: 802.11 standards, MCS index, Wi-Fi channels, channel bonding, signal thresholds, roaming parameters, WPA security, reason codes, PoE class and wattage, Ethernet and fiber pinouts, the OSI model.
  • Wi-Fi glossary. Searchable and offline, with both the IEEE and the Wi-Fi Alliance name for each term.
  • Laminated reference cards. The WLAN Pros laminated cards as PDFs, ready offline: quick-reference sheets and field checklists.
  • Network diagnostics. The active tools you run in the field: Ping, ping sweep, Traceroute, DNS lookup, WHOIS, BGP and ASN lookup, IP geolocation, port scan, SSL/TLS and HTTP inspectors, network discovery, Wake-on-LAN, packet sender, and Network Quality for latency, jitter, loss, and throughput.
On Android, the app also reads the live details of your connected Wi-Fi network and scans the Access Points around you: SSID, BSSID, RSSI, channel, channel width, band, and the 802.11 standard in use. iOS restricts most of this from third-party apps. Android does not, so the Android build surfaces it directly.

Free, and it stays out of your data

The whole app is free. No ads, no in-app purchases, no subscription. No account, no login. No analytics, no tracking, no advertising. WLAN Pros runs no server that receives your data. The calculators and reference run on your device. The diagnostic tools contact only the hosts you choose. Read the full privacy policy.

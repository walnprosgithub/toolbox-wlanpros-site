# Speed Test Services logos

Wordmark logos for the Speed Test Services reference page. Keyed by service
slug; the manifest-gated resolver (`lib/data/speedtest_logos.dart`) prefers SVG,
falls back to PNG, and degrades to a name-initial label when neither is present,
so the screen renders cleanly with any subset of these present.

Wordmarks only. No "certified" seals, no endorsement marks. Trademark/logo use is
comparison-context only and not yet legally cleared (see the research brief
limitations); Orb specifically warrants a courtesy confirmation given Keith's
relationship before its logo ships.

Expected files (either `<slug>.svg` preferred, or `<slug>.png`):

- ookla
- fast-com
- cloudflare
- mlab-ndt
- nperf
- waveform
- librespeed
- speedof-me
- orb
- openspeedtest
- speedtest-cli-iperf3
- isp-branded
